### Job Description
```
**PROJECT OVERVIEW**

I need a talented 3D animator or motion graphics designer to create a luxury brand intro video for MONTREZ, a high-end streetwear brand.

**ANIMATION SEQUENCE (3-5 seconds total):**

1. Scene 1 (0-1s): Ornate iron gates slowly swing open
2. Scene 2 (1-2.5s): Camera zooms forward, revealing elegant lake between gates and chateau
3. Scene 3 (2.5-4s): Camera crosses lake, approaching chateau
4. Scene 4 (4-5s): Chateau doors dramatically open (majestic, welcoming)
5. Scene 5 (5s): Smooth fade to white → website loads

**STYLE REFERENCE:**
• Luxury fashion aesthetic (Dior, Chanel, Louis Vuitton brand films)
• French Renaissance chateau architecture
• Golden hour lighting (warm, soft, luxurious)
• Film grain texture (subtle)
• Colors: Deep blacks, rich golds, cream/ivory tones

**TECHNICAL REQUIREMENTS:**
• Resolution: 1920x1080 (Full HD minimum)
• Frame Rate: 60fps (cinematic smooth motion)
• Format: MP4 (H.264 codec)
• File Size: MUST be <500KB (critical for web performance)
• Audio: Optional subtle luxury music OR no audio (preferred for autoplay)
• Poster Image: 1920x1080 JPG of first frame (<100KB)

**DELIVERABLES:**
1. Final video: montrez-intro-final.mp4 (<500KB, 1080p, 60fps)
2. Poster image: montrez-intro-poster.jpg
3. Source files: Blender/C4D/AE project (for future edits)

**WHAT I'LL PROVIDE:**
• 17-page detailed specification document (storyboard, lighting, colors, camera movements - every detail covered)
• Brand assets (logo, color palette)
• Visual references (chateau architecture, luxury brand examples)
• Technical export settings

You will have EVERYTHING needed to execute this perfectly. No guesswork.

**TIMELINE:**
• Day 1-2: Initial draft/preview (50% checkpoint)
• Day 3-4: Revisions based on feedback
• Day 5: Final delivery + optimization

**BUDGET:**
$125-150 USD (includes 1-2 rounds of revisions, source files)

**IDEAL CANDIDATE:**
✅ Experience with luxury brand videos (fashion, jewelry, high-end products)
✅ Portfolio shows smooth camera movements and cinematic quality
✅ 3D animation skills (Blender, Cinema 4D, 3ds Max) OR advanced After Effects compositing
✅ Understands web video optimization (compression without quality loss)
✅ Can deliver <500KB file size (critical requirement)
✅ Communicates clearly and delivers on time

**NOT A GOOD FIT IF:**
❌ You only do 2D motion graphics (this needs 3D realism)
❌ Your portfolio shows bright/cartoonish styles (we need luxury/cinematic)
❌ You can't deliver <500KB file size
❌ You need more than 7 days

**TO APPLY:**

Please include in your proposal:
1. Portfolio link (2-3 relevant examples of luxury/cinematic work)
2. Your approach: 3D animation, stock compositing, or hybrid?
3. Estimated timeline (be realistic)
4. Confirmation you can achieve <500KB file size
5. Any questions about the specification

**BONUS POINTS FOR:**
• Showing a similar chateau/mansion animation you've done
• Experience with French Renaissance architecture
• Understanding of luxury fashion brand aesthetics
• Quick turnaround (2-3 days possible)

I'm ready to award this project immediately to the right candidate. Full specification document will be shared upon hiring.

Looking forward to seeing your work!
```

## RESPONSE EVALUATION CHECKLIST

### When freelancers respond, evaluate using this:

**Portfolio Review:**
- [ ] Shows luxury/high-end brand work (NOT cheap/cartoonish)
- [ ] Camera movements are smooth (NO jitter or jarring cuts)
- [ ] Has done similar architectural/cinematic work
- [ ] Video quality is professional (good lighting, composition)
- [ ] Examples are compressed appropriately (not huge files)

**Communication:**
- [ ] Responds within 24 hours
- [ ] Asks smart questions about brand identity
- [ ] Confirms they can hit <500KB file size
- [ ] Provides clear timeline estimate
- [ ] Professional tone (not generic copy-paste)

**Technical Capability:**
- [ ] Confirms 3D software expertise (Blender/C4D) OR AE compositing
- [ ] Mentions specific compression techniques
- [ ] Offers source files (or willing to provide)
- [ ] Understands web video optimization

**Pricing & Terms:**
- [ ] Within budget ($100-150 range)
- [ ] Includes 1-2 revision rounds
- [ ] Clear deliverables list
- [ ] No hidden fees or upcharges

**Red Flags (Don't Hire):**
- ❌ Generic response (clearly copy-paste)
- ❌ Asks for full payment upfront (no milestones)
- ❌ Can't commit to file size requirement
- ❌ Portfolio doesn't match luxury aesthetic
- ❌ Timeline >7 days (too slow for 4s video)
- ❌ Terrible English (if communication is critical)
- ❌ No portfolio or only 2D work shown
