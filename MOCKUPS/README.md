## 🎨 Quick Reference: Visual Identity

### Brand Colors
- **Primary:** `#000000` (Black)
- **Secondary:** `#FFFFFF` (White)
- **Accent:** `#666666` (Grey)
- **Hover:** `#999999` (Light Grey)

### Typography
- **Display:** Bebas Neue (headings, nav, CTAs)
- **Body:** Inter (text, descriptions)

### Animation Philosophy
- **Speed:** Fast but visible (300-600ms)
- **Easing:** Smooth, not robotic
- **Purpose:** Every animation reinforces intent

## 🚀 Implementation Workflow

1. **Start with specs:**
   - Set up colors from `COLOR_PALETTE.md`
   - Apply typography from `TYPOGRAPHY.md`
   - Configure animations from `ANIMATIONS.md`

2. **Build layouts:**
   - Reference `ASCII_MOCKUPS.md` for structure

   - Study reference sites
