TECHNICAL RECOMMENDATIONS
================================================================================

ANIMATION LIBRARY: Framer Motion (primary)
- Excellent React integration
- Great DX for developers
- Good performance
- Active maintenance
- Bundle size: ~40KB gzipped

FALLBACK: GSAP + ScrollTrigger (for complex choreography)
- Industry standard
- Pixel-perfect control
- Timeline-based sequencing
- Premium plugins available
- Bundle size: ~65KB gzipped

PERFORMANCE TARGET:
- Lighthouse score: ≥90
- First Input Delay: <100ms
- Frame rate: 60fps (smooth animations)
- Core Web Vitals: all green

ACCESSIBILITY REQUIREMENTS:
- WCAG AA compliance (4.5:1 text contrast)
- Respect prefers-reduced-motion media query
- Keyboard navigation support
- Screen reader compatibility

MOBILE OPTIMIZATION:
- Disable parallax on mobile (performance)
- Simplify animations on low-end devices
- Touch-friendly interactions
- Test on: iPhone SE 2, Moto G7
